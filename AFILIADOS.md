## Cómo activar un afiliado nuevo

1. Consigue el enlace de afiliado
2. Edita `data/tools.json` — cambia `affiliateUrl` y pon `affiliateStatus: "active"`
3. Ejecuta `npm run build && npm run deploy:cloudflare`
